# PLI 

Réseau social basé décentralisé basé sur la géolocalisation. Projet de Bachelor à l'ETNA.

  - NodeJS + MySQL server
  - React Native Android+iOS

### Installation
##### Client

Voici comment installer les modules
```sh
$ cd Client
$ npm install 
```

Pour le lancer
```sh
$ react-native run-ios
$ react-native run-android
```
